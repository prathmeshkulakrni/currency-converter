# Currency Converter 🌍💱

A real-time currency converter web app that fetches live exchange rates and converts between 150+ currencies. Built with HTML, CSS, JavaScript, and a free exchange rate API.

[![Live Demo](https://img.shields.io/badge/Live-Demo-blue)](https://your-live-url.com)
[![License](https://img.shields.io/badge/License-MIT-green)](LICENSE)

## ✨ Features

- ✅ Real-time exchange rates from reliable API
- ✅ 150+ world currencies with flags
- ✅ Swap currencies with one click
- ✅ Responsive design (mobile + desktop)
- ✅ Clean, modern UI
- ✅ Instant conversion results
- ✅ Copy converted amount
